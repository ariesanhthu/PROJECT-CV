\section{Tổng quan}

\subsection{Mục tiêu}
Báo cáo này tổng hợp lại toàn bộ kết quả phân tích từ notebook
\texttt{test/foodseg103\_dataset\_analysis.ipynb}, bao gồm:
\begin{itemize}[leftmargin=1.5em]
    \item kiểm tra toàn vẹn dữ liệu train/test,
    \item thống kê kích thước ảnh và số object mỗi ảnh,
    \item phân tích mất cân bằng class,
    \item minh họa giải mã mask segmentation,
    \item đề xuất hướng sửa đổi quy trình huấn luyện.
\end{itemize}

\subsection{Nguồn dữ liệu và phạm vi}
\begin{itemize}[leftmargin=1.5em]
    \item Dataset: FoodSeg103 (103 class foreground).
    \item Tệp metadata: \texttt{meta.json}.
    \item Thư mục annotation: \texttt{train/ann} và \texttt{test/ann}.
    \item Thư mục ảnh: \texttt{train/img} và \texttt{test/img}.
\end{itemize}

Các biểu đồ trong báo cáo được tạo lại từ các tệp CSV đã xuất bởi notebook:
\texttt{foodseg103\_file\_index.csv}, \texttt{foodseg103\_summary.csv},
và \texttt{foodseg103\_class\_distribution.csv}.

\subsection{Thành phần báo cáo}
Báo cáo gồm 3 phần:
\begin{enumerate}[leftmargin=1.5em]
    \item Tổng quan,
    \item Phân tích và xử lí dữ liệu,
    \item Kết luận.
\end{enumerate}
