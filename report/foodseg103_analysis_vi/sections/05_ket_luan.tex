\section{Kết luận}

Báo cáo đã tổng hợp toàn bộ kết quả phân tích từ notebook FoodSeg103 theo dạng tài liệu LaTeX tiếng Việt, có tách file rõ ràng và kèm đầy đủ hình trực quan.

Kết luận chính:
\begin{itemize}[leftmargin=1.5em]
    \item Dataset có chất lượng tốt, độ toàn vẹn cao, độ phủ class đầy đủ.
    \item Mất cân bằng class theo long-tail là vấn đề trọng tâm cần xử lý khi huấn luyện mô hình segmentation.
    \item Có khác biệt kích thước ảnh giữa train và test, nên chuẩn hóa tiền xử lý để tăng tính ổn định khi đánh giá.
\end{itemize}

Với các đề xuất sửa đổi đã nêu, bộ dữ liệu có thể được khai thác hiệu quả hơn cho các mô hình phân đoạn đa lớp như DeepLabV3 hoặc các backbone lightweight.
